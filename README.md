# Game List

## Sobre o projeto

Game List é uma API desenvolvida durante o Intensivão Java Spring, ministrado pelo Prof. Nélio Alves (DevSuperior) entre os dias 8 e 12 de maio/2023. O programa exibe listas de jogos separados por categoria e exibe uma curta descrição sobre cada jogo na lista. Ao clicar no jogo, o programa exibe uma descrição completa e todos atributos do jogo.
